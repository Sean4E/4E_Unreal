Understanding Unreal Engine 5

Before you build anything, get familiar with the engine, its terminology, and how the editor works.

00

Unreal Engine Concepts

What is Unreal Engine 5? A real-time 3D creation engine used across games, film, architecture, and interactive media. It is free to download and use. UE5 introduced two breakthrough technologies: Lumen (dynamic global illumination) and Nanite (micro-polygon geometry). For this project, you will use UE5 to build a cinematic environment and render a short film from it.

Key Terminology

TermWhat It Means
LevelYour 3D world — everything you see in the Viewport lives in the Level
ActorAny object placed in a Level (a light, a mesh, a camera — all Actors)
ComponentA building block attached to an Actor (e.g. a Static Mesh component, a Light component)
BlueprintUE5's visual scripting system — you will not need to write any C++ code
ViewportThe big 3D window where you see and navigate your Level
OutlinerA list of every Actor in your Level (like a scene hierarchy)
Details PanelShows properties of the selected Actor — position, rotation, scale, and all component settings
Content BrowserYour file explorer inside the engine — where materials, meshes, textures, and sequences live

Editor Layout

When you open a UE5 project, you will see four key areas:

  • Viewport (centre) — your 3D scene. This is where you place and move objects.
  • Outliner (top-right) — a flat list of every Actor in the Level.
  • Details Panel (bottom-right) — properties for whatever Actor is selected.
  • Content Browser (bottom) — all project files, assets, and materials.
Think of the Viewport as your canvas, the Outliner as your layer list, the Details Panel as a property inspector, and the Content Browser as your file manager. All four work together.

Navigation Controls

Moving around the Viewport is essential. Here are the controls you will use constantly:

  • Right-click + WASD — fly through the scene (hold right mouse button and use keyboard)
  • Scroll wheel — zoom in/out
  • Alt + left-click drag — orbit around a point
  • Alt + right-click drag — dolly (zoom smoothly)
  • Middle-click drag — pan the view
  • F key — focus on the selected object
Tip: Hold right-click and scroll the mouse wheel to change your fly speed. This is helpful in large environments where default speed feels too slow.
Milestone: You understand the UE5 editor layout and can name the four key panels. You know that everything in a Level is an Actor.
01

Create Your Project

Every UE5 project starts in the Epic Games Launcher. We will use the Film / Video & Live Events category because our goal is a cinematic showcase, not a playable game.

  1. Open the Epic Games Launcher
  2. Click the Unreal Engine tab on the left sidebar
  3. Click Launch next to Unreal Engine 5 (install it first if you haven't)
  4. In the Unreal Project Browser, select Film / Video & Live Events
  5. Choose the Blank template
  6. Set the project type to Blueprint (not C++)
  7. Set Quality to Maximum Quality
  8. Tick Starter Content (gives you free materials and meshes)
  9. Leave Raytracing OFF (Lumen handles lighting for us)
  10. Set Target Platform to Desktop
  11. Name the project CinematicShowcase
  12. Click Create
Be patient: UE5 takes a few minutes to create a new project and compile shaders. You will see a loading screen — this is normal. Do not close the application.

First Look at the Editor

Once the project opens, take a moment to identify each panel:

  • Centre: the Viewport — you should see the default level with a floor, a table, some chairs
  • Top-right: the Outliner — lists all Actors in the scene
  • Bottom-right: the Details Panel — click any object and its properties appear here
  • Bottom: the Content Browser — expand it by clicking the tab
Tip: If any panel disappears, go to Window in the top menu bar and re-enable it.

Organise Your Content

In the Content Browser, create folders to keep your project tidy:

  1. Right-click in the Content Browser → New Folder → name it Materials
  2. Create another folder called Models
  3. Create another folder called Sequences
Milestone: You have a UE5 project open with Starter Content enabled. You can identify the Viewport, Outliner, Details Panel, and Content Browser.
02

Build Your Environment

Time to construct your cinematic space. You can build anything — an interior room, an outdoor landscape, a sci-fi corridor, a gallery. Start simple and add detail as you go.

Clear the Default Scene

  1. In the Outliner, select all default objects (click one, then Ctrl+A to select all)
  2. Press Delete to remove them
  3. You should now see an empty grey void in the Viewport

Create a Floor

  1. Go to Place Actors panel (left side, or Window → Place Actors)
  2. Under Basic, find Cube and drag it into the Viewport
  3. In the Details Panel, set Location to X: 0, Y: 0, Z: 0
  4. Set Scale to X: 20, Y: 20, Z: 0.1 (creates a large, thin floor)
  5. Rename it in the Outliner: double-click → type Floor
You should now see: A large flat surface in your Viewport. This is the ground plane of your environment.

Add Walls (Optional — for interiors)

If you are building an interior scene, add walls the same way:

  1. Drag another Cube from Place Actors into the scene
  2. Position it at the back edge of your floor
  3. Scale it to be tall and wide but thin: e.g. X: 20, Y: 0.1, Z: 4
  4. Duplicate with Alt+drag — hold Alt and drag the move gizmo to create a copy
  5. Rotate and position duplicates to form your walls

Place Assets

Now populate your environment. You have several options:

  • Starter Content: In the Content Browser, navigate to StarterContent/Props or StarterContent/Architecture. Drag meshes directly into the Viewport.
  • Fab / Quixel Bridge: Open Window → Quixel Bridge (or visit fab.com). Browse thousands of free photorealistic 3D assets. Download and drag into your scene.
  • Custom Models: Drag FBX or OBJ files from your file explorer into the Content Browser. UE5 imports them automatically. Then drag from Content Browser into the Viewport.

Transform Tips

  • W — Move tool (translation)
  • E — Rotate tool
  • R — Scale tool
  • Press G to toggle grid snapping on/off (helps align objects precisely)
  • Alt + drag — duplicate an object while moving it
  • End key — snap an object down to the surface below it
  • Ctrl + G — group selected Actors together

03

Materials & Surfaces

What is a Material in UE5? A node-based shader graph that defines how a surface looks. Unlike a simple colour picker, a UE5 material connects nodes together to control every aspect of appearance — colour, shininess, roughness, glow, and more.

Think of a Material as a recipe. The nodes are ingredients (colour, texture, number). The connections are the cooking instructions. The result is how the surface appears under light.

Create a Simple Material

  1. In the Content Browser, navigate to your Materials folder
  2. Right-click → Material
  3. Name it (e.g. M_Floor)
  4. Double-click to open the Material Editor
  5. You will see a single node called the Material Result with many input pins
  6. Right-click in empty space → search Constant3Vector → add it (this is a colour)
  7. Double-click the Constant3Vector to pick a colour
  8. Drag from its output pin to the Base Color input on the Material Result
  9. For Metallic: add a Constant node (single value), set it 0–1, connect to Metallic pin
  10. For Roughness: add another Constant, set it 0–1, connect to Roughness pin
  11. Click Apply (top-left) then Save

Key Material Inputs

InputControlsValues
Base ColorThe surface colour or textureRGB colour or texture node
MetallicIs it metal or non-metal?0 = plastic/wood, 1 = metal
RoughnessHow smooth or rough the surface is0 = mirror, 1 = chalk
NormalSurface detail without adding geometryNormal map texture
Emissive ColorMakes the surface glow (self-illuminating)RGB colour × intensity

Apply a Material

Simply drag the material from the Content Browser onto the mesh in the Viewport. Done. The surface updates instantly.

You should now see: Your object changes appearance immediately. If it looks too shiny, increase Roughness. If it looks flat, decrease Roughness.

Material Instances (Quick Variations)

Want the same material in different colours without rebuilding the node graph?

  1. Right-click your material in the Content Browser → Create Material Instance
  2. Open the instance — you will see exposed parameters (if you set them up in the parent)
  3. Adjust colour, roughness, etc. without touching the original
A Material Instance is like a preset. The parent material is the template; each instance is a variation. One template, many looks.

Emissive Materials (Glow Effects)

For neon signs, glowing panels, or screens:

  1. Open your material
  2. Add a Constant3Vector for the glow colour
  3. Add a Multiply node
  4. Connect the colour to Multiply A, add a Constant (value 5–50) to Multiply B
  5. Connect the Multiply output to the Emissive Color pin
  6. Apply and Save — the object now glows
Tip: Emissive materials work beautifully with Lumen. The glow will actually cast light onto nearby surfaces automatically.

Quick Approach: Use Starter Content

If the Material Editor feels overwhelming, just use the materials that came with your project:

  • In Content Browser, go to StarterContent/Materials
  • You will find wood, metal, brick, glass, and more — all ready to drag and drop
Milestone: Your environment has materials applied. Surfaces have colour, roughness, and metallic variation. No more default grey checkerboard.

04

Lighting with Lumen

What is Lumen? UE5's dynamic global illumination system. It simulates how light bounces off surfaces in real time — no baking required. Light hits a red wall and the nearby floor gets a subtle red tint. This happens automatically.

Traditional engines require you to "bake" lighting — a slow process that pre-calculates light bounces. Lumen does this in real time, so you can move a light and instantly see the result. It is like having a photography studio that updates lighting in real time.
Good news: Lumen is ON by default in new UE5 projects. You do not need to enable anything.

Light Types

Go to Place Actors → Lights to add lights to your scene. Here are the four types:

Set Up Your Lights

Your assignment requires a minimum of 3 light sources. Here is a suggested setup:

  1. Drag a Directional Light into the scene — this is your key light (the sun)
  2. Rotate it to set the angle (try X: -40, Y: -60, Z: 0 for a late-afternoon feel)
  3. Add a Point Light — position it near a focal point (above a table, inside a lamp)
  4. Add a Spot Light — aim it at a hero object to draw the viewer's eye
  5. Optionally add a Rect Light for soft fill or to simulate a window

Mood & Atmosphere Tips

  • Warm vs Cool: Warm lights (orange/yellow, 2700K–3500K) feel cosy and intimate. Cool lights (blue/white, 5500K+) feel modern or clinical.
  • High Contrast: A strong key light with dark shadows creates drama. Reduce fill lights for a cinematic look.
  • Coloured Fills: Add subtle coloured point lights (a faint blue in the shadows, a warm rim behind a subject) for richness.
  • Less is more: Beginners tend to over-light. Start dark and add lights selectively.

Post Process Volume

For additional cinematic control, add a Post Process Volume:

  1. Place Actors → search Post Process Volume → drag into scene
  2. In the Details Panel, tick Infinite Extent (applies to the whole level)
  3. Expand Exposure — set Min/Max Brightness for consistent look
  4. Expand Bloom — add subtle glow to bright areas (Intensity 0.3–0.8)
  5. Expand Color Grading — adjust Saturation, Contrast, and colour tints
Tip: For a filmic look, try setting Exposure → Metering Mode to Manual and adjusting the Exposure Compensation slider until your scene has the right brightness.
Milestone: Your environment has mood. At least 3 light sources with different types and colours. Lumen is bouncing light naturally. You should see coloured light bleeding onto nearby surfaces.
05

Cameras & Composition

Your cinematic needs multiple camera angles. UE5 uses Cine Camera Actors — cameras designed specifically for filmmaking, with real-world lens settings like focal length, aperture, and focus distance.

Add a Camera

  1. Go to Place Actors → Cinematic → Cine Camera Actor
  2. Drag it into the Viewport
  3. A camera icon appears in the scene with a preview window
  4. Rename it in the Outliner (e.g. Camera_WideShot)

Position Using Pilot Mode

The easiest way to frame a shot:

  1. Right-click the camera in the Outliner → Pilot
  2. The Viewport now shows exactly what the camera sees
  3. Navigate freely (right-click + WASD) to frame your shot
  4. When happy, click Stop Piloting (top-left of Viewport) or press Eject
  5. The camera stays exactly where you left it
You should now see: A small picture-in-picture preview in the corner of the Viewport showing what this camera sees.

Composition Tips

  • Rule of Thirds: Place key elements at intersections of a 3×3 grid, not dead centre
  • Leading Lines: Use walls, corridors, or edges to guide the eye toward a focal point
  • Depth: Include foreground, midground, and background elements for a sense of space
  • Variety: Mix wide establishing shots, medium shots, and close-ups

Add Multiple Cameras

Your assignment requires minimum 3 camera shots. Suggested setup:

  1. Camera_WideShot — shows the full environment (24–35mm focal length)
  2. Camera_MediumShot — focuses on a section or feature (50mm)
  3. Camera_CloseUp — highlights a specific detail or object (85mm+, low f-stop for blur)

Pilot each one and frame the shot before moving to the next.

06

The Sequencer

What is Sequencer? UE5's built-in timeline editor for creating cinematics. Think of it like a video editing timeline — you place cameras on tracks, set keyframes for movement, and control which camera is active at each moment.

If your Level is a film set and your cameras are physical cameras on tripods, then Sequencer is the director calling "Cut to Camera 2!" and telling the camera operator when to pan, zoom, and move.

Create a Level Sequence

  1. From the top menu: Cinematics → Add Level Sequence
  2. Save it in your Sequences folder — name it CinematicShowcase_Seq
  3. The Sequencer panel opens at the bottom of the screen
  4. You should see an empty timeline with a playhead (the red vertical line)
You should now see: A timeline panel at the bottom of the editor with a red playhead, play/pause controls, and an empty track list on the left.

Add a Camera Cut Track

The Camera Cut track is the master controller that decides which camera is filming at any moment.

  1. In the Sequencer panel, click the + Track button
  2. Select Camera Cut Track
  3. A new track appears in the timeline
  4. Click the + Camera button on the Camera Cut track
  5. Choose your first camera (e.g. Camera_WideShot)
  6. Move the playhead forward in time, click + Camera again, choose your second camera
  7. Repeat for your third camera

Add Camera Tracks

To animate a camera's movement, you need its own track:

  1. Click + Track → Actor To Sequencer
  2. Select your camera from the list (e.g. Camera_WideShot)
  3. The camera appears as a track with sub-tracks: Transform, Camera Settings, etc.
  4. Repeat for each camera you want to animate

Keyframing Basics

Keyframes record a value at a specific moment in time. UE5 interpolates between them automatically.

  1. Select a camera track in the Sequencer
  2. Move the playhead to the start of where you want movement to begin
  3. With the camera selected in the Viewport, press S to add a keyframe at the current position
  4. Move the playhead forward (e.g. to frame 90)
  5. Move the camera to its new position in the Viewport
  6. Press S again — a second keyframe is created
  7. Press Play in the Sequencer to preview the animation
Alternatively: Enable Auto-Key mode (the key icon with a circle in the Sequencer toolbar). When active, any change you make to a selected Actor automatically creates a keyframe. Be careful — turn it off when you are done animating.

07

Animate Your Scene

Your assignment requires at least 1 animated element. This can be a moving camera (which you may already have from Step 6), but also consider animating objects and lights to bring your scene to life.

Animate an Object

Move a door, rotate a sculpture, raise a platform — any object can be animated in the Sequencer.

  1. In the Sequencer, click + Track → Actor To Sequencer
  2. Select the object you want to animate (e.g. a door, a prop)
  3. Expand its track to reveal the Transform sub-track
  4. Move the playhead to the starting frame
  5. Select the object in the Viewport, position it, press S to keyframe
  6. Move the playhead forward
  7. Move/rotate the object to its new state
  8. Press S to keyframe the new position
Example: A door that slowly swings open. Keyframe it closed at frame 60, rotated 90 degrees at frame 120. That gives a 2-second door swing at 30fps.

Animate Lights

Changing light intensity or colour over time creates mood shifts:

  1. Add a light to the Sequencer (+ Track → Actor To Sequencer)
  2. Expand the track → find Light Component → Intensity
  3. Click the + next to Intensity to add it as a keyframeable track
  4. Set keyframes at different times with different intensity values
  5. You can also keyframe Light Color for colour shifts

Smooth Interpolation (Curves)

By default, keyframes interpolate linearly (constant speed). For smoother, more natural movement:

  1. Right-click a keyframe in the Sequencer
  2. Select Key Interpolation → Auto
  3. This creates smooth ease-in and ease-out curves
  4. For more control, open the Curve Editor (click the curve icon in the Sequencer toolbar)
Linear interpolation is like a car going 0 to 60 instantly. Auto/smooth interpolation is like gently pressing the accelerator — the movement eases in and out naturally.

Animation Ideas

  • Camera dollies: Slow, smooth forward movement through a space
  • Camera orbits: Circling around a central object
  • Door opening: Reveals a new area during the sequence
  • Light fading in: Gradually illuminates a dark space
  • Object rotation: A sculpture slowly spinning on a pedestal
  • Colour shift: Lights gradually changing from warm to cool
Tip: Subtle animations are more effective than extreme ones. A slow 5-degree camera tilt is more cinematic than a wild 360-degree spin. Think film, not theme park ride.
Milestone: Your Sequencer has at least 1 animated element beyond camera cuts. Pressing Play shows smooth movement. Keyframes are set to Auto interpolation for natural motion.
08

Render Your Cinematic

Time to export your sequence as a video file. UE5 has a dedicated tool called Movie Render Queue — it produces higher quality output than the old "Render Movie" button.

Enable Movie Render Queue (if needed)

  1. Go to Edit → Plugins
  2. Search for Movie Render Queue
  3. Make sure it is Enabled (it should be by default in UE5)
  4. If you had to enable it, restart the editor

Render Steps

  1. Open your Level Sequence by double-clicking it in the Content Browser
  2. In the Sequencer, click the clapperboard icon (Movie Render Queue) in the toolbar
  3. Alternatively: top menu → Cinematics → Movie Render Queue
  4. Click + Render and select your Level Sequence
  5. Click Unsaved Config to open render settings

Setting Up the Output Format

  1. In the settings window, remove any default output (select it, click the − button)
  2. Click + Setting and add .exr Sequence or Apple ProRes (for high quality) or Command Line Encoder (for H.264 MP4)
  3. For the simplest MP4 workflow: add Command Line Encoder, which uses FFmpeg to produce an MP4
  4. Set Output Resolution to 1920 x 1080
  5. Set Frame Rate to match your Sequencer (30fps)
  6. For Anti-Aliasing: Spatial Sample Count of 1 is fast, 4 is smoother but slower
Simplest approach: If the Command Line Encoder feels complicated, render as an Image Sequence (PNG) and then combine the images into a video using free tools like DaVinci Resolve, or even the built-in Windows Video Editor.

Start the Render

  1. Close the settings window
  2. Back in the Movie Render Queue, click Render (Local)
  3. UE5 will play through your entire sequence, rendering each frame
  4. A progress bar shows the estimated time remaining
  5. When finished, find your output file in the Saved/MovieRenders folder of your project
Be patient: Rendering takes time, especially at high quality settings. A 10-second sequence at 1080p with 4x anti-aliasing might take 5–15 minutes depending on your hardware. Do not move or click during the render.

Capture a Screenshot

For your assignment screenshot:

  1. Pilot one of your cameras (right-click → Pilot)
  2. Press F9 or go to the Viewport menu → High Resolution Screenshot
  3. Set the multiplier (2x gives you 4K from a 1080p viewport)
  4. Click Capture
  5. The screenshot is saved to Project/Saved/Screenshots
Milestone: You have a rendered MP4 video file (or image sequence) and at least one high-resolution screenshot. Open the video to verify it looks correct — check for smooth camera movements and correct lighting.
✓

Final Checklist

Before You Submit

  • Environment has a clear mood and atmosphere
  • 3D assets are imported or placed (not just an empty room)
  • Minimum 3 light sources with Lumen global illumination active
  • Level Sequence created with a Camera Cut track
  • Minimum 3 camera shots (different angles and compositions)
  • Smooth camera movements (keyframes with Auto interpolation)
  • At least 1 animated element (object, light, or camera movement)
  • Rendered via Movie Render Queue
  • Output resolution: 1920×1080 minimum
  • Frame rate: 30fps minimum
  • Output format: MP4
  • Screenshot captured (high-res)
  • Process document written (300–500 words)

Process Document Prompts

Your 300–500 word process document should cover:

  • What mood or atmosphere were you trying to create?
  • What assets did you use and where did they come from?
  • How did you approach lighting? What types did you use and why?
  • How did you compose your camera shots? Why those angles?
  • What was the most challenging part?
  • What would you change or improve with more time?
